# Library-Management-System
*Final Exam CINF201*


-This project is a Library Management System to be used by admin. It has components such as: adding a book, issuing a book to a customer, displaying available books, displaying issued books, viewing who has checked out a book including check-out and return date, incorporates a user-friendly menu, database storage in an SQLite Database, and is easy to interact with. 


-The code is made up of python language and imports SQLite. To compile and run, ensure the files library.db and library.py are both in directory folder "LibraryFinalExam201" and type python3 library.py. Once that command is executed a main menu will appear for the Library Management System, and the components above will be executable.
